Direct Indexing Portfolio Builder

Upload three CSV files to build a blended direct indexing portfolio. The app is flexible about column names and accepts common alternatives.

Accepted file types

You can upload files for:

  • S&P 500
  • Nasdaq-100
  • Russell 3000

Accepted column names include: ticker, symbol, stock and market_cap, market cap, marketcap, cap.

Build portfolio

Tips

  • You do not need exact header names.
  • Files can use common alternatives like symbol instead of ticker.
  • Market cap values can include commas or dollar signs.
  • If an upload fails, the error message should tell you exactly what to fix.
